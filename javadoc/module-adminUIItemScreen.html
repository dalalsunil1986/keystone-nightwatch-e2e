adminUIItemScreen

This module defines the commands an e2e test client may execute against keystone's item edit form screen. When using the assertElement* commands, you may specify one of the predefined element selector properties by prefixing it with the '@' sign or you can specify a string representing your own element selector. See usage example, usage example
