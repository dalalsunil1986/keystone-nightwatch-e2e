Module: adminUIListScreen

This module defines the commands an e2e test client may execute against keystone's Admin UI List Screen. A test client should try to avoid accessing the elements directly but instead use the commands to accomplish the testing. If a command is not available for the test purpose please submit a PR for consideration.

(inner) assertItemFieldUINotPresent(fields)

Asserts that the specified item field's UI is not present in the browser page. This command calls into the assertListScreenFieldUINotPresent command of the field under test, which gets passed the config.field.options.
Parameters:
Name Type Description
fields Object The array of fields to assert the UI precense on.
