adminUIApp

This module defines the commands an e2e test client may execute against keystone's Admin UI main app. When using the assertElement* or clickUIElement commands, you may specify one of the predefined element selector properties by prefixing it with the '@' sign or you can specify a string representing your own element selector. See usage example.
